04

Comunicazione MQTT

Topic sottoscritti dall'ESP:
lab_iot/mafogani/threshold
  → aggiorna soglie pianta (JSON)

lab_iot/mafogani/start-stop
  → comandi  START  /  STOP
Payload soglie (esempio):
{
  "name": "monstera_albo",
  "thresholds": {
    "temp":  { "min": 18, "max": 27 },
    "hum":   { "min": 60, "max": 75 },
    "light": { "min": 300,"max": 500}
  }
}
Con STOP l'ESP sospende l'invio dati · con START riprende — senza riavvio
05

Backend – FastAPI + InfluxDB

EndpointFunzione
GET /api/piante/soglie/{pianta}Soglie della pianta selezionata
GET /api/piante/syncmqtt/{pianta}Pubblica soglie sul broker MQTT → ESP
GET /api/piante/startstop/{cmd}Invia START / STOP all'ESP
GET /api/piante/data/{pianta}/{range}Serie storica da InfluxDB
GET /api/piante/latestdata/{pianta}Ultimo campione disponibile
Conversione ADC → kLux gestita server-side  ·  formula empirica KY-018: Lux = 500 / R_LDR_kΩ
